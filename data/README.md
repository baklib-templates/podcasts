- `Company`: 公司名
- `Main_Domain`: 公司主域名（不带协议）
- `Podcast_URL`: 播客栏目页/订阅页 URL（若未知可先填播客名称）
- `Episodes`: 集数（可用 `50+` 这种估算）
- `Last_Update`: 最近更新日期（建议填实际抓取日期或最近一集发布日期，格式 `YYYY-MM-DD`）
- `Main_Themes`: 主要话题（建议用 `;` 分隔，便于后续解析）
- `Has_Dedicated_Podcast_Section`: 是否有独立播客栏目页（`Yes/No`）

文件：`company_podcasts_template.csv` 可直接用 Excel 打开编辑。
